# 1. TodoModelクラスを作成する

Todoのデータを格納するためのModelクラスを作成します。Todoのデータ構造は次の通りです。

- id: Todoを一意に表すID
- title: Todoの内容
- isCompleted: Todoが完了してるかどうか(true＝完了)

ModelクラスはTypeScriptのクラスで定義します。

:pencil2: **app/models/todo.model.ts**
```ts
export class Todo {
  constructor(public id:number,
              public title:string,
              public isCompleted:boolean) {
  }
}
```
